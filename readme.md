# Engelsystem TranslationMapper

Small plugin to rewrite shown texts programmatically.
Be it Daemons, Trolls, Fairies or just normal Helpers, you can change them all without touching the translation files.

Replacements can be changed by editing [`src/Translator.php`](src/Translator.php).

## Installation

Copy or clone this repo as `resources/plugins/TranslationMapper` to your Engelsystem installation.
